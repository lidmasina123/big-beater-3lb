# big-beater-3lb
a 3lb combat robot with 4 x4 drive train and a weapon maximaized for inersia 
This is my 3 lb combat robot, based on a proven and competitive design. It features powerful 2306 drive motors, a 4-wheel-drive drivetrain, and a 6S battery, giving it excellent speed, traction, and control.
The robot is equipped with a dual-disk weapon that is designed to maximize rotational inertia by concentrating as much weight as possible near the outer edges of the disks
